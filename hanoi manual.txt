NA ROBOTU:
- UR+, provjeri moze li se gripper rucno otvarati/zatvarati
  - ako ne moze, Installation > URCaps > OnRobot Setup, stisni Refresh tipku, probaj opet rucno pomicati gripper
- gore Open > Installation > mihic.installation
- postavi TCP -> Installation > General > TCP -> TCP_hanoi
- postavi IP adresu -> gore desno sandwich menu > Settings > System > Network -> IP Address: 192.168.1.5 (nakon rada vratiti na IP adresu koja pise na robotu - 192.168.40.14)

NA LAPTOPU:
- pip install math3d pymodbus
- pip install git+https://github.com/jkur/python-urx.git
- postavi static ip address laptopa na 192.168.1.6, gateway 255.255.255.0
